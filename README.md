# aiBrowserPlagin

**AI-саммаризатор веб-страниц и PDF — браузерное расширение (Manifest V3).**
Один клик (или выделение фрагмента) → сжатая сводка текущей страницы во
всплывающем окне. Обработка идёт **прямым вызовом LLM-API из расширения по
вашему ключу (BYOK)** — без бэкенда, без сервера, без хранилища ключей на чужой
стороне.

> Простой быстрый саммаризатор произвольных страниц и PDF. Ваш ключ, ваш
> провайдер, ноль инфраструктуры.

## Чем это НЕ является (важно)

Это **не Shelf** и не его модуль. [Shelf](../shelf) — отдельный крупный проект:
локальная «умная библиотека» книг и заметок (Tauri-десктоп, SQLite, RAG с
цитатами, граф заметок). `aiBrowserPlagin` намеренно **гораздо у́же**: просто
саммаризатор веб-страниц/PDF — **без** библиотеки книг, **без** графа заметок,
**без** локальной БД, эмбеддингов и RAG. Не путайте два проекта.

## Что умеет (MVP)

**Фаза 1:**
- Саммаризация всей страницы одним кликом.
- Настройки BYOK: OpenAI-совместимый base URL, ключ, модель (хранятся в
  `chrome.storage` браузера).

**Фаза 2:**
- Поддержка PDF, открытых в браузере (через `pdf.js`).
- Саммаризация выделенного текста через контекстное меню.
- Боковая панель (Chrome Side Panel API) для длинных сводок.

Полное видение — [`docs/PLAN.md`](docs/PLAN.md); технический план и разбивка по
этапам — [`docs/TECHNICAL_PLAN.md`](docs/TECHNICAL_PLAN.md); за пределами MVP —
[`docs/POST_MVP_PLAN.md`](docs/POST_MVP_PLAN.md).

## Стек

- **TypeScript** (strict), **Manifest V3** (Chrome/Chromium).
- **Vite** + `@crxjs/vite-plugin` — сборка расширения.
- `pdf.js` (`pdfjs-dist`) — извлечение текста из PDF (Фаза 2).
- LLM — прямой `fetch` к OpenAI-совместимому endpoint. Бэкенда нет.

## Быстрый старт (разработка)

```bash
npm install
npm run build        # tsc --noEmit + vite build → dist/
# Chrome → chrome://extensions → включить Developer mode →
# Load unpacked → выбрать папку dist/

npm run dev          # dev-сборка с HMR (@crxjs)
npm run typecheck    # только проверка типов
```

BYOK: ключ вводится в настройках расширения и хранится в `chrome.storage`
браузера. Ключ **никогда** не коммитится в репозиторий.

## Требования

- **Node.js** 18+ и npm.
- Chrome/Chromium для загрузки распакованного расширения.
- Свой ключ к OpenAI-совместимому API (или локальный endpoint вроде
  Ollama/LM Studio на `http://localhost:11434/v1`).
